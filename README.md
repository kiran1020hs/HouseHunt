
# HouseHunt - Full Stack MERN Application

## Installation Steps

1. Install Node.js (LTS)
2. Install MongoDB OR use MongoDB Atlas
3. Open project in VS Code

## Backend Setup
cd backend
npm install

Create .env file (use .env.example)

npm start

## Frontend Setup
cd frontend
npm install
npm start

## Run Both Together
From root folder:
npm install
npm run dev

Backend runs on: http://localhost:5000
Frontend runs on: http://localhost:3000
